# **App Name**: UBS Life Moments Sprint Hub

## Core Features:

- Dashboard Navigation: Sidebar navigation allowing users to easily switch between different competency bands (A1-H1) and sprint days.
- Responsive Card Display: Displays analytical content (e.g., Problem Statement, Stakeholder Map, Success Criteria) within responsive 'Cards' for easy readability and modularity.
- Situational Analysis View (A1): Dedicated section for displaying the Problem Statement, visualizing the Stakeholder-Map (Power-Interest Matrix), listing HMW-Questions, and defining SMART-Goals.
- Success Criteria Table (B1): Presents a tabular view of success criteria, including Goal, Criterion, Type (Quali/Quanti), Measurement Method, and Thresholds (Min/Ziel/Stretch).
- Future Sprint Day Placeholders: Empty, pre-styled sections for upcoming sprint days (Ideation & Prioritization, Prototyping Logbook, Test Results, Pitch Deck) to maintain a consistent dashboard structure.
- SMART Goal Generation Tool: An AI-powered tool that assists users in drafting measurable, achievable, and relevant SMART goals based on initial objectives, focusing on topics like digital deposit signatures and WG-budgeting.

## Style Guidelines:

- Primary brand red: #E10600 (RGB: 225, 6, 0), used for accents, active navigation states, and key highlights.
- Main background color: #F4F4F4 (RGB: 244, 244, 244), providing a clean, light foundation for content.
- Accent and text color: #000000 (RGB: 0, 0, 0), used for primary text, headings, and strong contrasting elements.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, clean, and highly readable characteristics, suitable for a professional dashboard environment.
- Utilize minimalist line icons for navigation and card elements to maintain a modern and uncluttered aesthetic, aligning with the corporate branding.
- Implement a flexible, responsive layout using CSS Grid and Flexbox, featuring a persistent sidebar navigation and a main content area structured with 'Cards' for different analyses, adapting seamlessly across device sizes.
- Subtle, quick transition animations for sidebar expansion/collapse and card interactions to enhance user experience without distraction.
